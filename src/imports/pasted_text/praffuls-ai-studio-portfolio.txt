CREATE A COMPLETE PREMIUM PORTFOLIO WEBSITE DESIGN IN FIGMA

PROJECT NAME

PRAFFUL'S AI-POWERED STUDIO

OBJECTIVE

Design a world-class portfolio website for a creative studio specializing in e-commerce brand visuals, Amazon listings, Myntra creatives, product videos, brand stores, and e-commerce websites.

The website should feel like a premium creative agency rather than a freelancer portfolio.

DESIGN DIRECTION

Create a visual experience inspired by the best modern Framer portfolio websites and luxury editorial design.

The overall aesthetic should feel like a hybrid between:

• Luxury fashion editorial
• Creative technology studio
• Premium design agency
• Modern e-commerce creative partner

Focus on:

• Oversized typography
• Strong visual hierarchy
• Large whitespace
• Premium art direction
• Editorial composition
• Modern layouts
• Asymmetrical visual balance
• Conversion-focused design
• High-end creative presentation

Avoid:

• Generic SaaS layouts
• Startup website aesthetics
• Corporate agency templates
• Busy interfaces
• Overuse of gradients
• Stock-photo design styles

COLORS

Background: #F8F8F6

Primary Text: #111111

Accent Color: #F4C542

LAYOUT SYSTEM

• Desktop-first design
• 12-column grid
• Max width: 1440px
• Content width: 1280px
• Large section spacing (160px–220px)
• Strong vertical rhythm
• Premium whitespace throughout

TYPOGRAPHY

Headings:
Bebas Neue or similar condensed bold font

Body:
DM Sans or Inter

Typography should feel bold, modern, editorial, and premium.

IMPORTANT EDITABILITY RULES

Every image area must be created as a separate image frame placeholder.

Every video area must be created as a separate video placeholder frame.

Do NOT flatten any content.

Do NOT convert layouts into single images.

All content must remain fully editable.

All text layers must remain editable.

All image frames must allow direct image replacement.

All video placeholders must allow direct video replacement.

All cards should be editable components.

All buttons should be editable components.

All spacing and layout elements should remain editable.

The entire design must be organized using clean Figma layers and auto-layout where appropriate.
